# 使用引导

这份引导适合第一次使用 OCR Assistant 的用户。你可以把它当成一条完整流程：登录系统，上传 PDF，等待识别，查看 Markdown 结果，在知识库里复用文档，并用关键词或 AI 对话继续分析内容。

## 1. 登录系统

1. 打开系统地址。
2. 输入管理员或普通用户提供的用户名和密码。
3. 登录后进入主界面。左侧或顶部通常用于上传、任务列表、知识库和 AI 对话等操作。

如果登录失败，先确认账号、密码是否正确；如果是新账号，确认管理员已经把账号加入系统配置。

## 2. 上传 PDF

1. 在上传区域选择一个或多个 PDF 文件。
2. 选择 OCR 引擎：
   - MinerU：默认选择，适合版面复杂、表格较多、页数较长的材料。
   - PaddleOCR：适合较快地处理普通扫描件或文字版式相对简单的材料。
3. 点击开始处理。

系统只处理 PDF 文件。如果文件不是 PDF，请先转换成 PDF 后再上传。

## 3. 查看任务进度

上传后，文件会进入任务列表。每个任务会显示当前状态：

- 等待处理：文件已经进入队列，还没有开始识别。
- 处理中：系统正在读取页数、提交 OCR 或生成 Markdown。
- 处理完成：可以查看识别结果、分页内容和下载文件。
- 处理失败：可以查看错误信息，必要时换一个 OCR 引擎或重新上传。

批量上传时，任务会按队列顺序处理。长文档或版面复杂的文档需要更长时间。

## 4. 查看识别结果

任务完成后，可以打开结果查看区：

1. 点击任务或知识库里的文档。
2. 查看系统生成的 Markdown 内容。
3. 切换分页视图，按原 PDF 页码查看每一页的识别文本。
4. 如果结果满足要求，可以下载 Markdown 文件保存到本地。

Markdown 是后续检索、复制、整理和 AI 分析的主要文本格式。

## 5. 使用知识库

处理完成的文档会保存到知识库。知识库的用途是避免重复处理，并让你可以随时重新打开历史文档。

常见操作：

- 在知识库列表里查看已经完成的文档。
- 用文件名或正文关键词搜索文档。
- 打开文档后继续查看 Markdown、分页内容或下载结果。
- 上传同一份文件且 OCR 配置没有变化时，系统会优先复用已有结果，减少等待时间。

每个账号默认只能看到自己的文档；系统内置的公共文档对所有账号可见，新账号登录后也能看到。

## 6. 提取关键词

如果你需要快速定位某些主题，可以使用关键词提取功能。

1. 打开已完成的任务或知识库文档。
2. 在关键词输入框中输入关键词，多个关键词可以用逗号、分号或换行分隔。
3. 选择匹配方式：
   - 任意匹配：命中其中一个关键词就返回结果。
   - 全部匹配：同一段或同一页必须同时包含所有关键词。
4. 选择提取粒度：
   - 段落：适合阅读上下文和整理摘录。
   - 页面：适合按页码回到原文核对。
5. 点击提取，查看命中的页码、关键词和上下文。

如果结果太多，可以增加关键词或改用全部匹配；如果结果太少，可以减少关键词或改用任意匹配。

## 7. 使用 AI 对话

系统配置了大模型后，可以使用 AI 对话辅助整理材料。

适合提问的内容包括：

- “请总结这份材料的主要内容。”
- “请提取和青年工作有关的政策要点。”
- “请把这些段落整理成时间线。”
- “请按主题归类这些关键词命中结果。”

AI 回答可能需要人工核对。用于正式材料时，请回到原文页码或 Markdown 内容确认。

## 8. 下载和保存结果

处理完成后可以下载 Markdown 文件。建议按以下方式管理结果：

- 文件名保留原 PDF 名称，方便回溯。
- 重要材料保留原 PDF 和 Markdown 两份文件。
- 对关键摘录记录来源页码，便于后续核查。

## 9. 常见问题

### 上传后一直排队

可能是前面还有大文件正在处理。批量任务会按顺序执行，请等待当前任务完成。

### OCR 失败

可以尝试重新上传，或切换另一个 OCR 引擎。如果 PDF 页数很多、文件过大或扫描质量较差，处理时间会更长，也更容易失败。

### 识别结果有错字

OCR 会受扫描清晰度、字体、表格和页面倾斜影响。正式引用前请核对原 PDF。

### 找不到历史文档

先在知识库里搜索文件名或正文关键词；如果仍然找不到，确认当前登录账号是否与上传账号一致。

### 新账号为什么也能看到这份引导

这份《使用引导》是系统公共文档，不属于某一个个人账号。它会出现在所有账号的知识库中，用来帮助第一次接触系统的用户快速上手。
